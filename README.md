# Sistema-de-gestion-de-pacientes
Segundo proyecto
Este proyecto se enfoca principalmente en JavaScript, en él se presenta un HTML con una lista de pacientes en formato de tabla para un centro de nutrición llamado "Buena vida".
El HTML posee una sección script con varios links a las distintas funciones que otorga el sistema incluido un botón para agregar pacientes, el calculo de su IMC (índice de masa corporal), busqueda de pacientes utilizando expresiones regulares, eliminación de un paciente al hacer doble click sobre él y la carge de pacientes desde un servidor externo por medio de requisiciones AJAX.
